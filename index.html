What is HTML?

HTML stands for HyperText Markup Language. It was created for web browsers to easily read. Below are the basic HTML tags required for all web pages. This makes perfect sense to a web browser:

<!DOCTYPE html><html><head><title></title></head><body></body></html>

Just like the different languages that we speak and write, HTML has to follow proper "grammar" rules as well to make sense.

This is referred to as syntax.

Dive back in to Sublime Text

  1. Try changing something, like the text within the <p></p> tags.
  2. Save your file using File > Save, or use keyboard shortcuts (CMD + S or CONTROL + S)
  3. Right click and select Open in Browser.
  4. Change something else and open it in your browser again, just to make sure you've got it!
  5. After saving, you can also just refresh your browser to see the changes (rather than opening your file in a new tab every time).

Find a work flow that works best for you when making frequent changes. I like to use keyboard shortcuts to save my files in the editor (CTRL/CMND + S). Then I open my file in the browser once and simply refresh it (CTRL/CMND + R) after saving my edits to view the changes in the browser.
