##Markdown example You can even write in Markdown as long as the `data-markdown` attribute is included. See the [documentation for markdown syntax](http://daringfireball.net/projects/markdown/). Both text and code blocks are supported at this time. Code blocks need to be denoted either by three backticks (`) or by tabbing the content in one indent (at least 4 spaces). Tabbing to create a code block won't work in this slide deck if your content begins tabbed in in your source code. You can specify the language of the codeblock by following the three backticks immediately by the language name.
